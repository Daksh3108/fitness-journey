# Daksh Avtani Fitness Journey Site

This is the first version of `myfitnessjourney.com`, built as a simple static website.

## Files

- `index.html`: the main website content.
- `styles.css`: the visual design.
- `script.js`: small smooth-scroll behavior.
- `assets/fitness-hero.png`: the hero image.

## How to open locally

Open this file in your browser:

`C:\Users\daksh avtani\fitness-journey-site\index.html`

## Beginner deployment plan

1. Create a GitHub account.
2. Create a new repository named `fitness-journey-site`.
3. Upload or push this project folder to that repository.
4. Create a Vercel account using GitHub.
5. In Vercel, choose `New Project`.
6. Import the GitHub repository.
7. Click deploy.

Vercel will give you a live URL first. Later, you can buy/connect the custom domain `myfitnessjourney.com`.

## When to use Supabase

Do not add Supabase in version 1. Add it later when the site needs stored data, such as:

- Contact form messages.
- Progress logs.
- Diet logs.
- Admin-only updates.
- A friend transformation tracker.
